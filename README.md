# NLP vs Transformer Comparison

## Project Overview

This project compares a traditional Natural Language Processing (NLP) model with a transformer-based model for sentiment analysis on movie reviews.

The traditional NLP model uses TF-IDF vectorization and Logistic Regression, while the transformer-based model uses DistilBERT from Hugging Face.

The application allows users to enter movie reviews and compare predictions generated by both models through a Streamlit web application.

---

## Objectives

The main objectives of this project are:

- To understand traditional NLP techniques
- To understand transformer-based NLP models
- To compare the performance of both approaches
- To analyze sentiment from movie reviews
- To build a simple real-time sentiment analysis application

## Observations

The traditional NLP model performed slightly better than the pretrained DistilBERT transformer model on the IMDb dataset.

This happened because the transformer model was used without fine-tuning specifically on IMDb movie reviews.

Traditional NLP models can sometimes outperform pretrained transformer models on domain-specific datasets when transformers are not fine-tuned on the target dataset.
